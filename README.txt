Programme développer par Raphael Ptit Haddad et Bessel Kobeissi

Etapes pour lancer le programme : 
	-> executer le fichier compile.sh
		-> ./compile.sh

	-> executer le fichier run.sh
		-> ./run.sh

Une fois le programme de lancer:
	-> Le programme propose les commandes suivantes: 
		->l: affichage des lists
		->c: ajout d'un chanson
		->a: ajout d'un album
		->+: ajout d'une chanson dans un album 
		->|: ajout d'un livreAudio 
		->p: ajout d'une playlist
		->-: suppression d'une playlist
		->s: sauvegarde des lists dans les fichiers xml
		->h: commande help

Vous pouvez voir directement le resultat de la sauvegarde avec la commande 's'
	-> Aller dans le dossiers files
		->Album.xml
		->Playlist.xml
		->Element.xml

	->Pour ouvrir ces fichiers télécharger XML viewer
